# fastapi-example
Example of Fastapi Python framework with in memory db

## Requirements
- Python 3.8.1
- virtualenv

## Dependencies
- fastapi
- pydantic
- uvicorn

## Setup
- virtualenv venv
- source venv/bin/activate
- pip install -r requirements.txt
- uvicorn main:app --reload

## API Documentation
- http://127.0.0.1:8000/docs
- http://127.0.0.1:8000/redoc
